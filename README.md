# fileman
Java util for file system


## build.gradle 
1. Set repositories
    ```gradle
    allprojects {
      repositories {
        ...
        maven { url 'https://jitpack.io' }
      }
    }
    ```
2. Set dependencies
    ```gradle
    dependencies {
      implementation 'com.github.avaj-java:fileman:0.4.2.1'
    }
    ```
